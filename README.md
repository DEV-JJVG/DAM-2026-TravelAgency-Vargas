Para instalar y probar el proyecto es importante que clonemos el repositorio desde la URL e implementarla en VsCode
o descargar el archivo .zip que contenga todo el código fuente de la apliación web y adjuntarlo a la carpeta xampp y htdocs para poder probarla en el servidor local
